HTML arrangement
-----------
Fisrt use some container to package the different area. Use

```
<div id = "xxx"> </div>
```

Then at CSS, use `<style>` to choose different style.

```
#xxx{
background-color:black;
color:white;
text-align:center;
padding:5px;
}
```
